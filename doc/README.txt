Most of the documentation for CPSSchemas currently lives in:
../../CPSDocument/doc

- UML diagram for CPSSchemas in source form, editable with
  Poseidon for UML Community edition: intro-cpsschemas.zuml.

- `Note on using CPSSchemas on top of a stock CMF or Plone:
  cmf-compatibilty.txt(html) <cmf-compatibilty.html>`_

- `HOWTO_make_epoz_output_xhtml.txt(html) <HOWTO_make_epoz_output_xhtml.html>`_
  CPS now uses FCKeditor by default so this documentation may not be of interest

- Architecture notes / specs. Probably obsolete:

  + `localvocabulary.txt(html) <localvocabulary.html>`_
